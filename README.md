# ⛳ Golf Shot Simulation

A modern, interactive golf simulation platform that combines realistic golf physics with immersive 3D gameplay. The project features a Python-powered physics engine, a premium web dashboard, and an engaging play mode where players naturally swing a virtual golf club.

## 🚀 Features

### 🎯 Golf Simulator

* Realistic golf ball physics
* Python backend for trajectory calculations
* 3D ball flight visualization
* Wind and environmental effects
* Carry, roll, total distance, and peak height metrics
* Multi-shot comparison
* Interactive analytics dashboard
* Replay and trajectory history

### 🏌️ Play Mode

* Fully animated 3D golfer
* Natural mouse-based swing controls
* Hold, drag, and release to swing
* Dynamic power meter
* Multiple golf clubs
* Ball-follow camera
* Realistic swing animations
* Practice range and golf course gameplay

### 📊 Analytics

* Shot statistics
* Carry distance
* Total distance
* Ball speed
* Club head speed
* Launch angle
* Spin rate
* Hang time
* Shot history
* Performance comparison

### 🌍 Environment

* Dynamic weather
* Wind speed and direction
* Temperature
* Humidity
* Day and night modes
* Different golf courses

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sujith2325/golf-shot-simulation.git
```

Navigate to the project:

```bash
cd golf-shot-simulation
```

Create a virtual environment:

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```

## 🎮 How to Play

### Simulator Mode

1. Select a golf club.
2. Adjust environmental conditions.
3. Run the simulation.
4. View trajectory and analytics.
5. Compare multiple shots.

### Play Mode

1. Choose a golf course.
2. Aim the golfer.
3. Click and hold the mouse.
4. Drag backward to perform the backswing.
5. Swing the mouse forward and release.
6. Watch the ball fly with realistic physics.
7. Complete the hole with the fewest strokes.
